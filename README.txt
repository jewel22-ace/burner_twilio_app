A simple application that allows you to make call without any sim card.
Using internet alone you can call any number anywhere in the globe without the help of a sim card.

Think of this applicaiton as a burner phone.
Inspired by the fact that burner are banned in India, I came up with this project.
This all became possible because of twilio client and twilio voice api.

This is a startup phase of this project more features will be added in future.
